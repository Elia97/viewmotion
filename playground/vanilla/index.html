What it does

viewmotion observes elements entering the viewport and applies a CSS class that triggers a smooth transition. No scroll events. No polling. One observer.

✦
IntersectionObserver

Single global observer for all animated elements.

⚡
Zero dependencies

No GSAP. No scroll libraries. Just the platform.

♿
Accessible

Respects prefers-reduced-motion out of the box.

Drop-in HTML API

Animate any element by adding a data-motion attribute. No JavaScript configuration required.

<h2 data-motion='{"preset":"fade-up"}'>
<p data-motion='{"preset":"fade-up","delay":100}'>
<ul data-stagger='{"delay":80}'>
<li data-motion='{"preset":"fade-up","once":true}'>
